Effective Hamiltonians & control engineering

1 · Schrieffer–Wolff transformation

Start with $H = H_0 + V$ where $V$ couples energy-distinct subspaces. Look for an anti-Hermitian $S$ such that

$$ H_\text{eff} = e^{-S} H e^S = H_0 + V + [V, S] + \tfrac{1}{2}[[V,S], S] + \ldots $$

has no off-diagonal block between the subspaces.

The recursion gives $S^{(n)}$ from $[H_0, S^{(n)}] = V_\text{od}^{(n-1)}$, with the off-diagonal part of the previous order. Up to fourth order:

$$ \begin{aligned} H_\text{eff}^{(2)} &= H_0 + \tfrac{1}{2}\,[V, S^{(1)}], \\ H_\text{eff}^{(4)} &= H_\text{eff}^{(2)} + \tfrac{1}{24}\,[[ [V,S^{(1)}], S^{(1)}], S^{(1)}]. \end{aligned} $$

For the typical cQED case $H_0 = \omega_c a^\dagger a + \tfrac{1}{2}\omega_q \sigma_z$ and $V = g(a^\dagger\sigma^- + a\sigma^+)$, the generator is

$$ S^{(1)} = \tfrac{g}{\Delta}(a^\dagger\sigma^- - a\sigma^+), \qquad \Delta = \omega_q - \omega_c, $$

and $[V, S^{(1)}] = 2 g^2 / \Delta\,\big(a^\dagger a\, \sigma_z + \tfrac{1}{2}\sigma_z\big)$ giving the dispersive Hamiltonian directly.

2 · Dispersive shift — from SW to $\chi$

$$ H_\text{disp} = \omega_c\,a^\dagger a + \tfrac{1}{2}\tilde\omega_q\,\sigma_z + \chi\,a^\dagger a\, \sigma_z, \quad \chi = \frac{g^2}{\Delta}, \quad \tilde\omega_q = \omega_q + \chi. $$

For a transmon (which is multilevel), use SW including the $|2\rangle$ level. The result is

$$ \chi_\text{transmon} = -\frac{g^2}{\Delta}\cdot\frac{\alpha}{\Delta + \alpha}, $$

Negative $\chi$ for $\Delta > 0$: a transmon shifts the cavity down when excited.

Connecting two qubits dispersively through one cavity gives a cavity-mediated $ZZ$:

$$ H_{ZZ} = J_\text{eff}\, \sigma_z^{(1)} \sigma_z^{(2)}, \qquad J_\text{eff} = \frac{g_1 g_2}{2}\left(\frac{1}{\Delta_1} + \frac{1}{\Delta_2}\right) - \frac{g_1 g_2 \alpha}{\Delta_1\Delta_2} \;\text{(transmon)}. $$

3 · Magnus expansion & RWA averaging

For time-dependent drives, go into a rotating frame and average. The Magnus expansion writes

$$ U(T) = \exp\!\Big[ \sum_{k\geq 1} \Omega_k(T) \Big], \quad \Omega_1 = -i\int_0^T H(t)\,dt, \quad \Omega_2 = -\tfrac{1}{2}\int_0^T\!\!\int_0^{t_1}[H(t_1), H(t_2)]\,dt_2\,dt_1. $$

Truncating at $\Omega_1$ gives the textbook RWA. Higher Magnus terms produce Bloch–Siegert-type shifts and corrections to gate fidelities.

For a Gaussian pulse with bandwidth $\sigma$ driving a qubit with anharmonicity $\alpha$, the DRAG correction comes out of a 2nd-order Magnus expansion and a Schrieffer–Wolff for the leakage subspace. M5 + M6 together expose both as composable helpers.

Magnus — when it fails

Magnus converges as long as $\int |H(t)|\,dt \lesssim 1$. In strong-drive limit (saturated Rabi), go to Floquet theory directly: diagonalise $H_F = H(t) - i\partial_t$ in a Fourier basis. This is the M5 stretch goal.

4 · ZZ crosstalk

Even at zero drive, two transmons coupled by a static $g$ acquire a static $ZZ$ residual:

$$ \zeta = E_{11} - E_{10} - E_{01} + E_{00} \approx \frac{2 J^2 \alpha}{\Delta(\Delta + \alpha)} , $$

where $J = g_1 g_2 / 2(1/\Delta_1 + 1/\Delta_2)$ is the effective direct coupling.

$\zeta$ is the dominant systematic error in transmon stacks. Mitigation strategies — flag for M8:

  • Detuning into a "magic point" where $\zeta = 0$.
  • Tunable couplers (transmon mediator) that turn off $g_\text{eff}$.
  • Echoed cross-resonance that cancels $\zeta$ over the gate duration.

5 · Catalogue of two-qubit gates

Every two-qubit gate the surface and color code need can be built from the Hamiltonians on the previous page. The recipe table below maps each gate to its driving mechanism.

Gate Mechanism Active Hamiltonian Where
iSWAP Resonant JC exchange $g(\sigma^+_1\sigma^-_2 + \text{h.c.})$ Resonant cavity bus, or direct exchange
CZ Adiabatic $|11\rangle\leftrightarrow|02\rangle$ $J\,\sigma^+_1\sigma^-_2$ + transmon $|2\rangle$ Tuneable transmons
CR (cross-resonance) Drive q1 at $\omega_2$ $\Omega_d\cos(\omega_2 t)\sigma_x^{(1)} \to ZX$ in dispersive limit Fixed-frequency transmons (IBM)
Parametric iSWAP Modulate $\Delta(t)$ at $\omega_1-\omega_2$ $J(t) = J_0 + J_1\cos(\omega_\text{mod}t)$ Flux-modulated transmons
Cavity-mediated $J_\text{eff}$ Dispersive virtual photon $\chi_1\chi_2/\Delta\,\sigma_z^{(1)}\sigma_z^{(2)}$ Both spins coupled to same bus
Bell pair via JC $\pi/2$ Rabi swap + readout $g\sqrt{2}$ between $|01\rangle$ and $|10\rangle$ JC bus
All these gates are simulable in LindbladTTN

Once M2+M3+M4 land, every Hamiltonian above is a one-line template call. Drive shapes for gate timing come from M6. Calibration sweeps to extract optimal pulse parameters are exactly what the steady-state and energy-level solver (M7) is for.

6 · In code: SW + Magnus utilities (M5, shipped)

import numpy as np
from lindblad_ttn.effective import (
    schrieffer_wolff, dispersive_shift, magnus_average,
)

# ---------- 1) Numerical χ from full diagonalisation ----------
# Build a JC Hamiltonian in (qubit ⊗ cavity) basis (small dense matrix).
sz = np.array([[1, 0], [0, -1]], dtype=complex)
sp = np.array([[0, 1], [0, 0]], dtype=complex)
sm = np.array([[0, 0], [1, 0]], dtype=complex)
Nc = 8
a = np.zeros((Nc, Nc), dtype=complex)
for n in range(1, Nc):
    a[n - 1, n] = np.sqrt(n)
adag = a.T.conj()
omega_q, omega_c, g = 5.0, 4.7, 0.05
H = (0.5*omega_q*np.kron(sz, np.eye(Nc))
     + omega_c*np.kron(np.eye(2), adag@a)
     + g*np.kron(sp, a) + g*np.kron(sm, adag))

chi_num = dispersive_shift(H, qubit_dim=2, cavity_dim=Nc, n_max=1)
chi_sw  = g**2 / (omega_q - omega_c)
print(f"chi (numeric)  = {chi_num:.5f}")
print(f"chi (g²/Δ)     = {chi_sw:.5f}")  # leading-order SW

# ---------- 2) Schrieffer-Wolff block-diagonalisation ----------
H0 = np.diag([0.0, 1.0])
V  = 0.05 * np.array([[0, 1], [1, 0]], dtype=complex)
H_eff = schrieffer_wolff(H0, V, projector_A=np.diag([1, 0]).astype(complex),
                         order=4)
# off-diagonal entry of H_eff is suppressed to O(V³)

# ---------- 3) Magnus order-2 time-averaging ----------
sx = np.array([[0, 1], [1, 0]], dtype=complex)
def H_t(t):
    return 1.0 * np.cos(2.0 * t) * sx
H_avg = magnus_average(H_t, period=np.pi, n_points=256, order=2)
# H_avg should be ≈ 0 (cosine averages to zero over a period)
